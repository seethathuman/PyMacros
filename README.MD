# PyMacros
## Introduction
PyMacros is a preproccesor for python using the `##` syntax.
Check test.py for examples.

## Usage
1. Clone the repository
```terminaloutput
git clone https://github.com/seethathuman/PyMacros.git
cd PyMacros
```

2. Run
```terminaloutput
./main.py test.py -o out.py -r
```
```terminaloutput
3.14159
16
12
```
